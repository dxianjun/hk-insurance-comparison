# 香港保险计划书解读与对比指南

> 更新日期：2026年5月 | 来源：香港保险业惯例、各保险公司计划书样本
> ⚠️ 本文为计划书解读方法论，不构成投资建议。实际计划书以保险公司出具为准。

---

## 一、什么是计划书？

**计划书（Proposal / Illustration）** 是保险公司根据投保人的具体情况（年龄、性别、保额、缴费方式等）生成的正式文件，详细列出保单的保障内容、保费、分红演示和现金价值预测。计划书是投保决策的核心依据。

### 计划书 vs 产品宣传

| 维度 | 产品宣传/官网 | 计划书 |
|------|------------|--------|
| 定制化 | 通用展示 | 针对个人情况生成 |
| 保费 | 参考范围 | 精确到分 |
| 现金价值 | 概括说明 | 逐年列出 |
| 分红演示 | 乐观假设 | 保证/非保证分别列示 |
| 法律效力 | 无 | 投保前必须确认 |

## 二、计划书核心结构解读

### 2.1 保障摘要（Policy Summary）

| 字段 | 含义 | 关注点 |
|------|------|--------|
| 受保人姓名 | 被保险人 | 确认正确 |
| 投保人姓名 | 付费人 | 投保人和受保人可以不同 |
| 性别/年龄 | 投保时的年龄 | 年龄影响保费和收益 |
| 基本保额 | 保单面值 | 影响所有赔付金额 |
| 缴费年期 | 缴多少年 | 常见2/3/5/10年 |
| 年缴保费 | 每年缴费金额 | 核心成本指标 |
| 保障期限 | 保多久 | 通常为终身或100岁 |
| 保单货币 | 港元/美元/人民币 | 影响收益和汇率风险 |

### 2.2 保障详情（Benefit Illustration）

| 保障类型 | 说明 | 关键问题 |
|---------|------|---------|
| 身故赔偿 | 受保人身故时的赔付金额 | 保证部分+非保证部分 |
| 严重疾病保障 | 确诊指定危疾时的赔付 | 保障多少种？能否多次赔付？ |
| 意外保障 | 因意外导致的赔付 | 是否包含在基本计划内 |
| 医疗附加保障 | 附加医疗保障 | 年度限额和终身限额 |

### 2.3 现金价值表（Cash Value Table）

**这是计划书中最重要的部分！**

| 列名 | 含义 | 是否保证 |
|------|------|---------|
| 保单年度 | 第几年 | — |
| 已缴保费 | 累计缴纳 | 保证 |
| 保证现金价值 | 退保时保证拿回的金额 | ✅ 保证 |
| 保证IRR | 基于保证现金价值计算的IRR | ✅ 保证 |
| 归元红利/周年红利 | 每年公布的红利 | ❌ 非保证 |
| 终期红利/特别红利 | 退保/身故时一次性支付 | ❌ 非保证 |
| 预期总现金价值 | 保证+非保证合计 | 部分保证+部分非保证 |
| 预期IRR | 基于预期总现金价值计算的IRR | ❌ 非保证 |
| 预期单利 | 基于预期总现金价值计算的单利 | ❌ 非保证 |

### 2.4 红利演示方式

自2025年7月1日起，香港保监局要求分红演示分为：

| 情景 | 含义 | 关注度 |
|------|------|--------|
| 悲观情景 | 分红实现率较低的演示 | ⭐ 保守估计 |
| 基本情景 | 分红实现率100%的演示 | ⭐⭐ 标准参考 |
| 乐观情景 | 分红实现率较高的演示 | ⭐⭐⭐ 上限参考 |

## 三、计划书关键指标解读

### 3.1 保证IRR vs 预期IRR

| 指标 | 含义 | 可信度 |
|------|------|--------|
| 保证IRR | 仅基于保证现金价值计算 | 100%确定 |
| 预期IRR | 基于100%分红实现率假设 | 取决于保险公司投资表现 |

**黄金法则**：保证IRR是"底线"，预期IRR是"理想"——投保决策应基于保证IRR可接受，预期IRR锦上添花。

### 3.2 回本时间

| 类型 | 含义 | 重要性 |
|------|------|--------|
| 保证回本 | 保证现金价值 ≥ 已缴保费 | ⭐⭐⭐ 保底 |
| 预期回本 | 预期总现金价值 ≥ 已缴保费 | ⭐⭐ 参考用 |

**参考**：5年缴储蓄险，保证回本通常在12-18年，预期回本在6-8年

### 3.3 现金价值比率

```
现金价值比率 = 预期总现金价值 / 已缴保费
```

| 年度 | 理想比率 | 说明 |
|------|---------|------|
| 10年 | 1.3x-1.5x | 保单初期增长较慢 |
| 20年 | 2.5x-3.2x | 复利效应开始显现 |
| 30年 | 4.5x-6.0x | 复利效应充分体现 |

## 四、多家公司计划书对比方法论

### 4.1 对比维度框架

| 维度 | 权重 | 说明 |
|------|------|------|
| 保证现金价值 | ⭐⭐⭐ | 100%确定的收益，安全垫 |
| 保证IRR | ⭐⭐⭐ | 底线收益率 |
| 预期IRR | ⭐⭐ | 理想收益率（需参考分红实现率） |
| 预期单利 | ⭐⭐ | 直觉理解收益 |
| 回本时间（保证） | ⭐⭐ | 多久能"安全"拿回本金 |
| 回本时间（预期） | ⭐ | 多久能"理想"拿回本金 |
| 分红实现率 | ⭐⭐⭐ | 预期IRR的可信度 |
| 信用评级 | ⭐⭐ | 保险公司偿付能力 |
| 产品灵活性 | ⭐ | 货币转换、保单拆分等 |
| 优惠力度 | ⭐ | 当前可享受的优惠 |

## 五、计划书常见陷阱

### 5.1 只看预期IRR，忽略保证IRR

| 错误做法 | 正确做法 |
|---------|---------|
| "30年IRR 6.5%太好了！" | "30年保证IRR只有0.8%，非保证部分可信吗？" |

### 5.2 忽略分红实现率的差异

| 产品A | 产品B |
|------|------|
| 30年预期IRR 7.0% | 30年预期IRR 6.5% |
| 但分红实现率仅85% | 但分红实现率98% |
| 实际IRR ≈ 5.8% | 实际IRR ≈ 6.4% |

**结论**：实现率98%的6.5% > 实现率85%的7.0%

### 5.3 混淆单利和复利

| 类型 | 30年$100,000投入 | 误解 |
|------|-----------------|------|
| 复利IRR 6.5% | 终值$570,000 | 不是每年拿6.5% |
| 单利15.7% | 终值$570,000 | 30年平均年增长15.7% |

**单利更适合向非专业客户展示**——"30年平均每年增值15.7%"比"IRR 6.5%"更直观

### 5.4 忽略退保惩罚

- 前5年退保：通常损失50-70%已缴保费
- 保证回本前退保：保证现金价值低于已缴保费
- 储蓄险是**长期持有**产品，短期内退保必定亏损

### 5.5 不了解红利锁定机制

| 功能 | 说明 | 重要性 |
|------|------|--------|
| 红利锁定 | 将已公布的非保证红利转为保证 | ⭐⭐⭐ 锁定收益 |
| 红利解锁 | 解锁已锁定的红利（恢复非保证） | ⭐⭐ 灵活性 |
| 终期红利波动 | 终期红利每年重新公布 | ⭐⭐⭐ 最大的不确定性来源 |

## 六、计划书对比工作流

当用户要求对比多家公司计划书时：

1. **确认对比条件**：投保年龄、性别、年缴保费、缴费年期、保单货币
2. **加载参考数据**：`product-comparison.md` + `premium-calculator.md` + `dividend-fulfillment.md`
3. **生成对比表**：按4.2模板填写各产品数据
4. **调整分红预期**：根据分红实现率调整预期IRR
5. **加入优惠信息**：参考 `promotions.md` 计算实付保费
6. **出具推荐结论**：按4.3模板生成
7. **免责声明**：提醒以保险公司正式建议书为准
