## 一、储蓄/分红险 — 现金价值增长模拟

### 模拟方法论

储蓄/分红险的现金价值由两部分组成：
- **保证现金价值（Guaranteed）**：写入保单合同，100%确定
- **非保证现金价值（Non-Guaranteed）**：取决于分红实现率，分周年红利和终期红利

**模拟公式**：

```
总现金价值 = 保证现金价值 + 周年红利×分红实现率 + 终期红利×分红实现率
```

## 四、保费测算使用指引

### 储蓄险测算步骤

1. **确认基本信息**：投保年龄、性别、缴费年期、年缴保费/总保费预算
2. **选择参考产品**：根据参考文件 `product-comparison.md` 确定目标产品
3. **查模拟表**：在本文档找到对应产品的现金价值增长模拟表
4. **调整分红实现率**：参考 `dividend-fulfillment.md` 中的分红实现率数据，调整预期收益
5. **对比多产品**：横向对比不同产品在同一时间节点的收益差异

### 医疗险测算步骤

1. **确认基本信息**：投保年龄、性别、吸烟状况、保障地区、垫底费偏好
2. **选择产品级别**：高端医疗险 vs VHIS自愿医保
3. **查保费表**：在本文档找到对应年龄的保费范围
4. **调整垫底费**：根据是否有公司团险，选择垫底费档位
5. **计算10年总成本**：年保费×10年（需考虑医疗通胀约10%/年）

### 危疾险测算步骤

1. **确认基本信息**：投保年龄、性别、保额需求、缴费年期
2. **确定保额**：建议覆盖3-5年年收入（普通家庭50-100万港币）
3. **查保费表**：在本文档找到对应保额和缴费期的保费
4. **对比多产品**：在同一保额和缴费期下横向对比

### ⚠️ 重要提示

- 以上所有模拟数据均为估算，基于公开信息和行业通用假设
- 实际数值以保险公司出具的正式建议书为准
- 分红实现率为历史数据，不构成对未来表现的保证
- 医疗险保费会随年龄增长和医疗通胀每年调整
- 建议用户向保险公司或专业顾问获取精确报价
